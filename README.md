### 1. Clone the repository

```bash

cd unit-converter-app
```

### 2. Install dependencies

```bash
npm install
```

## ğŸš€ Usage

Once the application is running:

- **Input a value** to convert.
- **Select the "From" and "To" units** using the dropdown menus.
